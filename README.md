# Silent Camera10 📷

iOS搭載のカメラアプリをウェブ上で完全再現した、サイレント撮影対応のカメラアプリです。

## ✨ 特徴

- **🎨 iOS 17風 UI**: iPhone標準のカメラアプリのような直感的な操作感と美しいダークモードデザイン
- **🔇 完全サイレント**: シャッター音なしで撮影が可能（Web技術を活用）
- **📲 スマホ保存**: 撮影した写真はブラウザ内に保存され、そのままスマホのアルバムにダウンロード可能
- **🖼️ ギャラリー機能**: 
  - アプリ内で撮影した写真を一覧表示
  - 長押しで複数選択・一括ダウンロード・一括削除
- **🔒 プライバシー**: 写真データは全てブラウザ内（LocalStorage）に保存され、サーバーへは一切送信されません
- **📱 PWA対応**: ホーム画面に追加することで、ネイティブアプリのように使えます

## 🛠️ 技術スタック

- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- WebRTC (getUserMedia API)
- Canvas API
- Web Share API

## 🚀 使い方

1. このサイトにアクセスし、カメラの使用を許可します。
2. 白い円形のシャッターボタンを押して撮影します。
3. 左下のサムネイルをタップすると、撮影した写真の一覧（ギャラリー）を確認できます。
4. ギャラリーで写真を長押しすると、一括ダウンロードや削除が可能です。
5. プレビュー画面から個別にシェアや削除も可能です。

## ⚠️ 注意事項

- iOSのSafariでは、仕様上、自動ダウンロードがブロックされる場合があります。「保存」ボタンを押した後に表示されるダイアログに従ってください。
- 保存容量はブラウザのLocalStorage制限（通常5〜10MB）に依存します。容量がいっぱいになった場合は、古い写真を削除してください。
